# 🎓 军事理论应知应会 · 刷题系统

移动端优化的军事理论题库练习系统，支持微信浏览器打开。

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 📖 **练习模式** | 随机打乱刷题，已做题不重复，支持题型筛选 |
| 📝 **模拟考试** | 可配置题目数量和考试时间，倒计时自动提交 |
| ❌ **智能错题本** | 记录错误次数、时间，支持按最新/高频/未复习排序 |
| ⭐ **收藏夹** | 标记重点题目，集中复习 |
| 🔍 **题目搜索** | 支持搜索题干和选项关键词 |
| 📊 **学习统计** | 正确率环形图、分题型统计、学习热力图、历史记录 |
| 🤖 **自动解析** | 每道题自动生成解析，标注高频错题警告 |
| 📱 **移动端优化** | 底部导航栏、触控友好的44px+按钮、响应式布局 |
| 📤 **数据导入/导出** | JSON格式备份，跨设备迁移数据 |
| 🔌 **离线可用** | PWA支持，可添加到手机主屏幕 |
| ⌨️ **键盘快捷键** | 桌面端支持键盘答题（A/B/C/D选择，←→切换判断题） |

## 🚀 部署方式

### 方式一：GitHub Pages（推荐）

1. Fork 或上传本仓库到 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，根目录 (`/`)
4. 保存后获得访问地址：`https://你的用户名.github.io/仓库名/`
5. 在微信中直接打开该链接即可使用

### 方式二：本地运行

无需 Node.js！直接用浏览器打开 `index.html` 即可。

### 方式三：任何静态托管

上传到 Netlify、Vercel、Cloudflare Pages 等均可。

## 💡 使用技巧

1. **微信中使用**：打开链接后，点击右上角"..."→"添加到桌面"，下次可从桌面图标直接进入
2. **数据备份**：定期在统计页面导出JSON备份，换设备时导入即可
3. **错题复习**：按"高频"排序错题，优先复习做错次数最多的题目
4. **键盘操作**：桌面端可使用键盘快捷键提高刷题速度
