二、简答题

1. Python是动态类型语言，请解释什么是动态类型，并与静态类型语言进行对比。在Python中，同一个变量名可以在程序运行期间被赋值为不同类型的数据，请举例说明这一特性，并简述这种特性的优点和潜在的注意事项。
答案：动态类型：变量类型在运行时确定，无需声明，同一变量可赋不同类型。如：x=10; x="hello"; x=[1,2,3]。静态类型（C/Java）在编译时确定类型，变量必须先声明。优点：灵活简洁，开发效率高。注意事项：类型错误运行时才发现；建议用Type Hints辅助；注意变量命名规范。
2. 请分别说明 if-elif-else 选择结构的执行逻辑，以及 for 循环和 while 循环的适用场景有何不同。在此基础上，解释什么是死循环，并说明如何避免死循环的发生。
答案：if-elif-else：从上到下依次检查，遇到第一个True即执行对应代码块并跳出剩余分支。for循环：适用于已知循环次数或遍历可迭代对象。while循环：适用于不确定循环次数、依赖条件控制的场景。死循环：循环条件永远为True。避免方法：确保条件最终变False、在循环体内修改条件变量、使用break、设置最大迭代次数。
3. 请说明在Python中进行文件操作的基本步骤，以及以 'r'、'w'、'a' 三种模式打开文件的区别。同时，解释异常处理机制中 try、except、finally 三个关键字各自的作用，并简述为什么要对文件操作进行异常处理。
答案：步骤：open()→读/写→close()，推荐用with open() as f:自动关闭。'r'只读（文件须存在）；'w'写入（清空原内容/创建新文件）；'a'追加（保留原内容/创建新文件）。try：放可能出错的代码；except：捕获并处理异常；finally：无论是否异常都执行（释放资源）。文件操作可能遇到文件不存在、无权限、磁盘满等异常，必须处理以保证程序不崩溃且文件正确关闭。
4. Python中常见的字符串格式化方式有哪三种(%-格式化、str.format()、f-string)？请分别说明它们的基本语法，并从可读性、灵活性和版本兼容性三个方面比较三者的优劣。
答案：%-格式化："我叫%s，%d岁"%(name,age)，Python2风格。format()："我叫{}，{}岁".format(name,age)，支持索引和关键字。f-string：f"我叫{name}，{age}岁"，Python3.6+。可读性：f-string > format > %。灵活性：format最灵活（支持复杂格式化）。兼容性：%最广（支持Python2），f-string仅3.6+。推荐用f-string。
5. 请解释面向对象编程中"封装"和"继承"两个核心概念的含义和目的。结合实例说明：封装如何提升代码的安全性？继承如何提高代码的复用性？并简述子类重写父类方法的意义。
答案：封装：将数据和操作数据的方法绑定在一起，隐藏内部实现。通过私有属性(__xxx)和公开getter/setter控制外部访问，可加入数据验证 → 安全性。继承：子类继承父类的属性和方法，公共代码写在父类 → 复用性。重写：子类重新定义父类方法，实现多态——同一接口不同行为。如Animal父类有make_sound()，Cat重写为"喵喵"，Dog重写为"汪汪"。
6. 请解释Python中切片操作的三个参数(start、stop、step)的含义，并分别举例说明：如何利用切片实现字符串的反转、如何取出字符串的偶数位字符。另外，说明字符串的 split() 和 join() 方法分别有什么作用，并举例。
答案：切片：[start:stop:step]。start起始索引(含)，stop结束索引(不含)，step步长(负数为反向)。反转："hello"[::-1] → 'olleh'（从右向左取）。偶数位："abcdef"[::2] → 'ace'（隔一个取一个）。split()：按分隔符拆分字符串为列表。如"a,b".split(",")→['a','b']。join()：用分隔符连接列表为字符串。如"-".join(['a','b'])→'a-b'。
